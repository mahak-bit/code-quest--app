# Code Quest

A gamified coding learning platform designed to help beginners practice programming concepts through interactive challenges and progress tracking.

## Features
- Interactive coding challenges
- Progress tracking
- Beginner-friendly interface
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- React

## Installation
1. Clone the repository
2. Run npm install
3. Run npm start

- User authentication
- AI-powered hints
- Leaderboard system
